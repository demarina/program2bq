Para crear este aplicacion web la he dividido en dos partes principalmente, una parte seria la perteneciente a la de restFul realizada en Java y otra parte la peticion de datos al usuario. Para poder realiar he tenido que usar dos servidores(esto en realidad debia haberlo hecho en uno solo, pero por rapidez y comodidad la mayor parte de la interfaz web me la he intentado ahorrar de hacer en Java) una para gran mayor parte web incluyendo los archivos html, css, javascript y php, php lo uso para el envio del mail de confirmacion. El index principal tiene graficamente muy buena pinta para mi gusto, lo demas ya no tanto porque he ido un poco pillado tiempo y como en la explicacion del programa viene especifido que no se va a prestar atencion en ese punto, esta primera parte esta corriendo encima de un servidor local montado con la aplicacion XAMPP, el segundo servidor que he usado ha sido Apache Tomcat que he añadido en eclipse y en el he metido la parte de restful que he hecho en Java. Lo primero que pide el servicio son nombre, mail e identificador, se requiere rellenar todos los campos, que compruebo con jQuery al igual que el mail sea un mail correcto (contenga . y @), una vez este todo correcto se envia un mail de confirmacion y si el mail se envia correctamente se consume el primer servicio restful, que servira para añadir al usuario si no existe ya en la BBDD y si existe notificar el error en el log. Una vez hecho esto y si va correcto todo se da aceptar y se redirige a index2, y paso en la uri el identificador para llevarlo almacenado y poder usarlo mas adelante. En esta parte de index2.html hay dos maneras de usarlo, una es mandar un mensaje, rellenando el campo mensaje y pulsando en mandar, si el campo mensaje esta vacio no se podra acceder al servicio restful y en el se almacenara el mensaje de ese identificador, el segundo servicio en esta pagina para ver los mensajes que hay ya. Todos estos servicios la informacion viaja en la uri y dependiendo de que servicio restful consumir pues se redirigira a una uri distinta. Para la parte restFul he usado Java, eclipse y Jersey sumado a Apache Tomcat para hacer correr todo en un servidor.
